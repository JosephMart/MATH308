Problem 2
Part A

y = (exp(t)*(t - 1) + 1)/t^2
y' = (exp(t) + exp(t)*(t - 1))/t^2 - (2*(exp(t)*(t - 1) + 1))/t^3
Subs into the 1/t^2 + (exp(t)*(t - 1))/t^2 results in 2*y(t) + t*((exp(t) + exp(t)*(t - 1))/t^2 - (2*(exp(t)*(t - 1) + 1))/t^3)
Using MATLAB's simplify command results in 2*y(t) + t*((exp(t) + exp(t)*(t - 1))/t^2 - (2*(exp(t)*(t - 1) + 1))/t^3) which is exactly
what is should be

Part B

The solution graph for t=0, y(t) approaches 0. The solution graph for large t's, y(t) approaches Inf
